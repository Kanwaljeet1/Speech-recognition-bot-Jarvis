# Jarvis Speech Recognition Bot

Jarvis is a speech recognition bot created with Python. It listens for the keyword **"Jarvis"** and then accepts voice commands to perform tasks.

## Requirements

- Visual Studio Code (VSCode) installed on your system
- Python installed
- A Python virtual environment set up in your system

## Setup and Running Instructions

1. **Open your terminal (in VSCode or system terminal).**

2. **Activate your virtual environment:**

   - On Windows:
     ```bash
     .\venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

3. **Run the Python script:**

   ```bash
   python jarvis.py
